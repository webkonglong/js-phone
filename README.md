# js-phone[根据手机号查询归属地]

### * 430570个号段，算是截至2019-06-04年最新号段
### * json文件为了减少储存,key最前面的那个1忽略了。


## 使用方法

### `const find = require('js-phone')`
### `console.log(find(18888888888))`

## 输出

### `{ p: '北京', c: '北京', o: '移动', zc: '100000', ac: '010', lc: '110100' }`

#### * `p` 省份
#### * `c` 城市
#### * `o` 运营商
#### * `zc` 邮政编码
#### * `ac` 区号
#### * `lc` 省市区号(就是身份证号码的前六位)
